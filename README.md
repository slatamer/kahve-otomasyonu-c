# ☕ C ile Kahve Otomasyonu (Konsol Uygulaması)

Bu proje, C programlama dilinin temel prensiplerini ve konsol tabanlı uygulama geliştirme süreçlerini pekiştirmek amacıyla geliştirilmiştir.

## Öne Çıkan Özellikler
- **Dinamik Menü:** ASCII karakterler ve `system("color")` ile zenginleştirilmiş kullanıcı arayüzü.
- **Güvenli Kodlama:** Visual Studio ortamında derleyici yapılandırmaları ve hata ayıklama süreçleri başarıyla yönetilmiştir.
- **Etkileşim:** `conio.h` kütüphanesi ile klavye girişi kontrolü sağlanmıştır.

## Teknik Detaylar
- **Dil:** C
- **IDE:** Visual Studio 2022
- **Konseptler:** Kontrol yapıları (`switch-case`, `if-else`), döngüler, standart I/O fonksiyonları.

## Nasıl Çalıştırılır?
1. `.sln` dosyasını Visual Studio ile açın.
2. `main.c` dosyasını derleyin.
3. Uygulamayı çalıştırarak kahve seçenekleri arasından seçim yapmaya başlayabilirsiniz.
